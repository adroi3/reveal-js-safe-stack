What is the SAFE Stack?

  • A cross-platform set of components for creating web applications in F#
  • Includes everything from the server up to the client
  • Stands for: Saturn Azure Fable Elmish
  • Components can be replaced
  • Dotnet CLI template exists

Saturn

  • Is a layer of abstraction on top of Giraffe
    • Framework for creating ASP.NET Core web applications in F#
  • Adds for example the possibility to use authentication

Fable

  • F# to JavaScript compiler
  • Designed to produce readable and standard JavaScript code
  • Enables access to JavaScript libraries

Elmish/Elmish React

  • Used for creating UIs
  • SAFE uses the Elmish React module
  • Uses the Model-View-Update pattern

Model-View-Update pattern

  • Model
    • State of the program
    • Is passed to the Update/view function
  • Program
    • Displays the data and has functionality in it (view/update function etc.)
    • Similar to a ViewModel + View
  • Init function
    • Starting point of a program
  • Update function
    • Changing the Model
    • No mutations only creates an updated version of the Model
  • View function
    • Describes the UI
  • Command
    • Fetches data from the server, writing data to the database etc.
    • Only option to cause side-effects
  • Message
    • Is used for determination of what to do in the update function
