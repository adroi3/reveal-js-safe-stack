What is the SAFE Stack?

  • A cross-platform set of components for creating web applications in F#
  • Stands for: Saturn Azure Fable Elmish
  • Components can be replaced
  • Includes everything from the server up to the client
  • Dotnet cli template exists

Saturn

  • Is a layer of abstraction on top of Giraffe
    • Framework for creating ASP.NET Core web applications in F#
  • Adds for example the possibility to use authentication

Fable

  • F# to JavaScript compiler
  • Designed to produce readable and standard JavaScript code
  • Enables access to JavaScript libraries

Elmish/Elmish React

  • Used for creating UIs
  • SAFE uses Elmish React
  • Uses the Model-View-Update pattern
